# A/B тестирование платежной системы

Статистический анализ эффективности новой механики оплаты.

## Описание проекта
Проведение A/B теста для оценки влияния новой платежной системы на ключевые бизнес-метрики. Принятие решения о запуске на всех пользователей.

## Методы анализа
- Z-тест для сравнения конверсии
- Бутстрап для ARPU, ARPPU, медианного платежа
- Анализ распределения платежей
- Сравнение активных и неактивных пользователей
- Статистическая значимость (p-value, доверительные интервалы)

## Ключевые результаты
- ARPPU: +34.7% (статистически значимо, p=0.0029)
- Медиана платежа: +94.9% (статистически значимо, p=0.0001)
- Конверсия: -0.46% (не значимо, p=0.4455)
- ARPU: +22.6% (не значимо, p=0.0980)

## Бизнес-инсайты
- Обнаружен новый популярный продукт за 1900 рублей в тестовой группе
- Эффект сильнее проявляется у активных пользователей
- Структура покупок сместилась в сторону премиум-сегмента
